# 🏥 SmartClinic

SmartClinic is a healthcare appointment and clinic workflow management system built with Laravel.

The platform helps patients book appointments online while enabling clinic staff to manage consultations, billing, receipts, and patient routing across departments.

---

## 🚀 Features

### Patient Features

* Online appointment booking
* Appointment status tracking
* Contact support form
* Mobile responsive interface

### Doctor Features

* View patient appointments
* Add doctor notes
* Add prescriptions
* Add treatment instructions
* Schedule next visit dates

### Front Desk Features

* Billing management
* Payment status updates
* Receipt generation
* Department routing

### Admin Features

* Dashboard analytics
* Staff registration
* Appointment approval
* Appointment rejection
* Appointment rescheduling
* Contact message management
* User role management
